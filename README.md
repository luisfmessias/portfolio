# Portfólio — Luís Fernando

Este é o meu portfólio pessoal como **Desenvolvedor Júnior Full Stack**, onde apresento meus projetos, habilidades e experiência prática com desenvolvimento web.

O objetivo deste projeto é demonstrar minha evolução técnica, cuidado com UI/UX, animações e boas práticas de código.

---

## Tecnologias utilizadas

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

---

## Funcionalidades

- Interface moderna e responsiva
- Animações suaves com Framer Motion
- Seção de projetos com foco técnico (UI, UX, performance)
- Layout otimizado para desktop e mobile
- Código organizado e componentizado

---

## Foco do projeto

- **UI**: design limpo, moderno e consistente  
- **UX**: navegação clara e intuitiva  
- **Performance**: componentes otimizados  
- **Clareza de código**: estrutura organizada e legível  


## Como rodar o projeto localmente

```bash
# Clone o repositório
git clone https://github.com/luisfmessias/portfolio.git

# Acesse a pasta
cd portfolio

# Instale as dependências
npm install

# Rode o projeto
npm run dev
